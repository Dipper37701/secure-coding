## Installation
### Download
```
git clone https://github.com/Dipper37701/secure-coding.git
```
### BackEnd
```
cd secure-coding
pip install -r requirements.txt
python backend/app.py
```
### FrontEnd
```
cd secure-coding/frontend
npm install
npm start
```
